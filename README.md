# Basic NodeJS APIs


Basic implementation of NodeJS Express and Websocket with Clients for testing

| Service               |         Url                | Port      |
|:---------------------:|:--------------------------:|----------:|
| express APIs (list)   |        /api/v1/list        |   3001    |
| websocket             |        /ws                 |   8077    |

## run Express client curl tests
```bash
bash clients/express/curl 
```
